Basic introduction to HTML and CSS

Hello people of the internet, if you are viewing this that means you are curious about how HTML and CSS work, good for you. Firstly in case you have no clue what Html(Hypertext Markup Language) or CSS (Cascading Style Sheet) are they are simple just the languages/components used for creating websites and codes. HTML focuses mainly on the actual build and layout of the code meaning where everything will go, CSS however aims more for the aestetics of the code so making everything look nice and how you want it to.

How html can access CSS

There are three main possible ways for HTML to access CSS: Inline, Internal and External.
Inline style
Inline style coding can be used if there is only a single event in a section of code, an example of this style is when a certain piece of style code such as the colour you use then placing that colour into a certain space this can be done with lines like; style="colour:blue;margin-right:20px;"
Internal style

Exrternal
The external sytle sheet can be implemented into many different pages at once. With an external style sheet you can change the look of an entire website just by changing a single file.







CSS box-model

The CSS box model is often used to show how CSS should be handled in a website by creating separate sections and setting them a certain type of data to contain.

Content- The content box holds all the basic content that is completely necessary and wants to be shown clearly.
Padding- A small gap held between the content and the border where unnecessary data will partially begin to fade out of the picture.
Border- The border is the solid space that will not show any of the content data that reaches this far.
Margin- The margin is usually set to be the edge of the page, so basically nothing should honestly even be placed in this gap.
