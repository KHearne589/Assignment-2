Basic introduction to HTML and CSS

Hello people of the internet, if you are viewing this that means you are curious about how HTML and CSS work, good for you. Firstly in case you have no clue what Html(Hypertext Markup Language) or CSS (Cascading Style Sheet) are they are simple just the languages/components used for creating websites and codes. HTML focuses mainly on the actual build and layout of the code meaning where everything will go, CSS however aims more for the aestetics of the code so making everything look nice and how you want it to.

How html can access CSS

There are three main possible ways for HTML to access CSS: Inline, Internal and External.
Inline style
Inline style coding can be used if there is only a single event in a section of code, an example of this style is when a certain piece of style code such as the colour you use then placing that colour into a certain space this can be done with lines like; style="colour:blue;margin-right:20px;"

CSS box-model

The CSS box model is often used to show how CSS should be handled in a website by creating separate sections and setting them a certain type of data to contain.

Content- The content box holds all the basic content that is completely necessary and wants to be shown clearly.
Padding- A small gap held between the content and the border where unnecessary data will partially begin to fade out of the picture.
Border- The border is the solid space that will not show any of the content data that reaches this far.
Margin- The margin is usually set to be the edge of the page, so basically nothing should honestly even be placed in this gap.

What is a scripting language?
One of the most basic and well known scripting languages that is used today is Javascript, developed by Netscape to give all other web developers the ability to design an interactive website. After Javascript was created it started to support all browsers as it was the only scripting language able to interact with HTML code and became an important part top our browsing experience, although we may not realise it Javascript helps us do everything from using links to other websites or simply watching a video on youtube it is all because of Javascript's ability to interact with HTML code.

IF and ELSE statements
Another basic part to scripting languages is allowing the code to be able to make decisions on its own.
IF statements are used in order to execute a certain piece of code but only if the specified condition is true.
ELSE statements however execute a certain piece if code if the condition is true but also executes a different piece of code for if the condition is false.
The last part I shall mention within this page for basic scripting is two kinds of loops. FOR and WHILE loops.
FOR loops
A loop is a piece of code that can be executed constantly over and over again until it fulfills a preset criterion. FOR loops will be executed while the count is less than or equal to five. Once count equals six the loop will move on to the next statement.
WHILE loops
The WHILE loops are used to repeat a block of code as long as a specified condition is true. If the condition is true, the code within the selected pair of brackets is executed, then the script checks the condition againand if the condition is still true, the selected piece of code is executed again. This process continues until the condition becomes false or you use a 'break' statement to end the loop. Example:
