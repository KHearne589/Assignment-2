Basic introduction to HTML and CSS

Hello people of the internet, if you are viewing this that means you are curious about how HTML and CSS work, good for you. Firstly in case you have no clue what Html(Hypertext Markup Language) or CSS (Cascading Style Sheet) are they are simple just the languages/components used for creating websites and codes. HTML focuses mainly on the actual build and layout of the code meaning where everything will go, CSS however aims more for the aestetics of the code so making everything look nice and how you want it to.
